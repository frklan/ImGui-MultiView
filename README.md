# MultiView

A playground to learn a little more advanced ImGui layouts.

## Compiling

Make sure the following tools are installed on you machine:

- C++ compiler that supports C++20.
- Conan
- CMake

Then execute:

```
git clone ...
cd multiview
take build
cmake .. --preset default
make -j
```
